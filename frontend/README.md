# Vite + Deno + React

## Installing Deno
You need to have Deno v1.28.0 or later installed to run this repo:

Linux/macOS:
```sh
$ curl -fsSL https://deno.land/x/install/install.sh | sh
```

Windows:
```powershell
irm https://deno.land/install.ps1 | iex
```

## Running

Start a dev server:

```
$ deno task dev
```

## Deploy

Build production assets:

```
$ deno task build
```
